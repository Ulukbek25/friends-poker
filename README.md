# Friends Poker MVP

Онлайн Texas Hold'em для друзей на виртуальные фишки.

## Запуск

```bash
npm install
npm start
```

Откройте http://localhost:3000

Для игры с друзьями через интернет разверните проект на Render/Railway/VPS. Обычный Netlify не подходит для постоянного Socket.IO-сервера.

## Реализовано
- приватные комнаты по коду
- до 8 игроков
- виртуальные фишки
- блайнды 25/50
- улицы preflop/flop/turn/river
- fold/check/call/raise/all-in
- автоматическое определение комбинации и победителя
- чат
- адаптация под телефон

## Ограничения MVP
- без регистрации и сохранения аккаунтов
- без side pots при нескольких all-in
- данные комнат хранятся в памяти сервера
