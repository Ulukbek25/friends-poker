# Покер у Улукбека

Онлайн Texas Hold'em для приватной игры с друзьями на виртуальные фишки.

## Возможности
- Приватные комнаты до 8 игроков
- Texas Hold'em: префлоп, флоп, тёрн, ривер, вскрытие
- Автоматические блайнды 25/50
- Красивые анимации раздачи и отображение комбинации победителя
- Чат стола
- Демо-экран MBANK QR для начисления виртуальных фишек

> QR является демонстрационным и не проводит реальных платежей. Денежные ставки, вывод средств и денежные выигрыши отсутствуют.

## Локальный запуск
```bash
npm install
npm start
```
Откройте http://localhost:3000

## Render
- Build Command: `npm install`
- Start Command: `npm start`
